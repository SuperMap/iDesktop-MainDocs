Shortcuts provided by  for Scene operations are listed below.

### Shortcut of the Object Operations

Shortcut | Description  
---|---  
K | Break  
U | Break exactly  
E | Extend  
T | Trim  
M | Move with the coordinate  
O | Offset  
H | Chamfer  
F | Fillet  
Y | Copy  
J | Join line  
Alt + A | Add the node  
Alt + D | Edit the node  
R | Rotate  
Alt + L | Draw the line to split  
Alt + R | Draw the region to split  
Alt + G | Group  
Alt + P | Resolve  
Alt + U | Union  
Alt + E | Erase  
Alt + I | Intersect  
Alt + F | Xor
