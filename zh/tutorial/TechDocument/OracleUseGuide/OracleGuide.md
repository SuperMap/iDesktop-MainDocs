---
Oracle 的安装及建库配置直接影响 Oracle 的运行性能，而 SuperMap SDX+ for Oracle Plus 的性能在很大程度上受
Oracle 的性能影响。因此，为提高 SuperMap SDX+ 的性能，在安装与配置 Oracle 数据库时，应该对各个参数进行优化，以提高
Oracle 的性能。下面主要介绍 Oracle 安装需要注意的事项、内存分配原则及各种优化参数的设置。
